## 📌 Key Core Topics Covered

### 1. Data Preprocessing & Feature Engineering
- **Exploratory Data Analysis (EDA):** Data profiling, correlation analysis, and data visualization.
- **Data Cleaning:** Handling missing data (Imputation) and outlier detection/removal.
- **Feature Scaling & Transformation:** Standardization, Normalization, and Log transformations.
- **Categorical Encoding:** One-Hot Encoding, Label Encoding, and Target Encoding.

### 2. Supervised Learning
- **Regression:** Simple & Multiple Linear Regression, Ridge, Lasso, and ElasticNet.
- **Classification:** Logistic Regression, Support Vector Machines (SVM), K-Nearest Neighbors (KNN), and Naive Bayes.
- **Tree-based Models:** Decision Trees, Random Forest, AdaBoost, Gradient Boosting, and XGBoost.

### 3. Unsupervised Learning & Dimensionality Reduction
- **Clustering:** K-Means, Hierarchical Clustering (Agglomerative), and DBSCAN.
- **Dimensionality Reduction:** Principal Component Analysis (PCA) and t-SNE.

### 4. Model Evaluation & Optimization
- **Metrics:** MSE, RMSE, R2-Score, Accuracy, Precision, Recall, F1-Score, and ROC-AUC.
- **Tuning:** Hyperparameter tuning using GridSearchCV and RandomizedSearchCV.
- **Validation:** K-Fold and Stratified K-Fold Cross-Validation.
